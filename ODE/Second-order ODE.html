We consider $y'' + by' + cy = 0$. We need to convert into a system of first-order ODEs by setting $u = y'$. Then we get

$$ u' = -b/a u - c/a y $$

and

$$ y' = u $$

In the code, we set the vector $X = [u \, y]^T$

def dX_dt(X, t=0):
    return np.array([ -b/a*X[0] -   c/a*X[1] ,
                           X[0] ])
